## Баг-репорт
### На запрос «Прочитай текст» камера смартфона не запускается
**Описание:** В приложении после перехода в чат с Алисой и запроса «Прочитай текст» выдаётся ответ «Не могу прочитать страницу», и камера не запускается.<br><br> **Шаги:**<br> 1. Перейти в настройки телефона и управление приложениями — убедиться, что для «Яндекс» разрешено использование камеры;<br> 2. Открыть приложение «Яндекс — с Алисой»;<br> 3. Сказать «Алиса» и выбрать «Чат с Алисой»;<br> 4. Сделать запрос «Прочитай текст»;<br><br>  **Ожидаемый результат:** Алиса отвечает «дайте посмотреть поближе» и запускает камеру. Фотография возвращается в диалог с Алисой. Есть голосовой и текстовый ответ, что текста на картинке не обнаружено. Под бабблом есть кнопки в разделе «Ещё информация по этой картинке».<br> **Фактический результат:** На запрос пишет: «Не могу прочитать страницу», не запускает камеру.<br><br> **Дополнительная информация:** Скриншот — https://imgur.com/a/PpcBi8r.
